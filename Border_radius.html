border-radius property can have 1-4 values:

border-radius with 1 value: this value will get apply for all the four corners.

border-radius with 2 value:first value applies to top-left and bottom-rigth corners, second values applies to top-right and bottom-left corners.

border-radius with 3 value:first value applies to top-left corner,second values applies to top-right and bottom-left corners and third value applies to bottom right corner

border-radius with 4 value:first value applies to top-left corner,second values applies to top-right corner,third value applies to bottom right corner and fourth value applies to bottom-left corner.
